## ✨ Features

- Spotify-inspired desktop layout
- Sidebar navigation
- Interactive song cards
- Hover play button
- Bottom music player
- Play / pause / next / previous controls
- Progress bar
- Volume control
- Search filtering
- Responsive structure for future expansion

---

## 🛠️ Built With

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**
