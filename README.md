# 🎧 Spotify Clone (HTML, CSS, JS)

A modern Spotify-inspired music player built with pure frontend technologies.

## 🚀 Features

- 🎵 Music player (play, pause, next, previous)
- 🟢 Spotify-style UI layout
- 📂 Sidebar navigation (Home, Search, Library)
- 🔍 Live search filtering
- 🎯 Hover play animations
- 📊 Progress bar + volume control
- 📱 Responsive design
- 🎨 Clean dark theme
